# axiom-website

Source code and assets for the official Axiom GNU/Linux website ([axiom.bottlecat.xyz](https://axiom.bottlecat.xyz)).

### `cat /home/axiom/website.txt`

This repo hosts the static landing page, documentation, wiki, download mirrors, and release notes for Axiom GNU/Linux.

Built with clean, vanilla HTML/CSS (no bloated frameworks) so it loads blazingly fast.

### Site Map

- `/` — Main landing page & interactive roadmap
- `/axm/` — AXM Package Manager documentation & source tags
- `/wiki/` — Axiom system wiki & guide
- `/downloads/` — ISO download mirrors
